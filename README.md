# PhpNet
PHP.net CLI search

## Usage

####Add permission to run it
```bash
$ chmod +x phpnet
```

####Running

```bash
$ ./phpnet <function>

#Ex.
$ ./phpnet preg_replace
```

## Run it globally
```bash
$ cp phpnet /usr/local/bin/phpnet
# Or
$ sudo ln -n phpnet /usr/local/bin/phpnet
```

####Now it can be called directly from anywhere
```bash
$ phpnet <function>
```
